🔸Git commit 規範

| fix  | 改bug    | fix: 修正登入錯誤訊息|
| style| 格式調整（不影響程式邏輯） | style:修改主色 |
| refactor| 程式重購（不影響功能）| refactor:把CSS改成Sass |
| docs | 現有文件變更 | docs: README新增oo事項 |
| chore | 開發工具、建構流程、設定等雜項 | chore:新增.gitignore檔名 oo |
